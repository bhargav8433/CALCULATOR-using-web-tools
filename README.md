# Calculator

A simple calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an easy-to-use interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Responsive and user-friendly design

## Technologies Used

- HTML
- CSS
- JavaScript

## Live Demo

https://bhargav8433.github.io/CALCULATOR-using-web-tools/

## How to Run

1. Clone the repository.
2. Open `index.html` in your web browser.
